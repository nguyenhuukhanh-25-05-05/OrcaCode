# Libraries — Open-Source UI Library Catalog

> **Module:** Reference  
> **Read Time:** As needed  
> **When:** During implementation — pick 2+ libraries

---

## CORE UI FRAMEWORKS

| Library | Install | Best For |
|---------|---------|----------|
| **shadcn/ui** | `npx shadcn-ui@latest init` | Tailwind components, full source code |
| **daisyUI** | `npm install daisyui` | Tailwind CSS components (pure CSS) |
| **Headless UI** | `npm install @headlessui/react` | Unstyled accessible primitives |
| **Mantine** | `npm install @mantine/core @mantine/hooks` | Full React component library |

## ANIMATION LIBRARIES

| Library | Install | Best For |
|---------|---------|----------|
| **Framer Motion** | `npm install framer-motion` | React animations, layout, gestures |
| **GSAP** | `npm install gsap` | Professional timeline animations |
| **AOS** | `npm install aos` | Scroll-triggered animations (easy) |
| **Anime.js** | `npm install animejs` | Lightweight JS animations |

## ICON LIBRARIES

| Library | Install | Best For |
|---------|---------|----------|
| **Lucide React** | `npm install lucide-react` | Clean, consistent, ISC license |
| **Heroicons** | `npm install heroicons` | By Tailwind team |
| **Phosphor React** | `npm install phosphor-react` | 6 style variants per icon |

## DATA & CHARTS

| Library | Install | Best For |
|---------|---------|----------|
| **Chart.js** | `npm install chart.js` | Simple canvas charts |
| **ECharts** | `npm install echarts` | Feature-rich charts, 3D |

## UTILITIES

| Library | Install | Purpose |
|---------|---------|---------|
| **Zustand** | `npm install zustand` | Lightweight state management |
| **React Hook Form** | `npm install react-hook-form` | Performant forms |
| **date-fns** | `npm install date-fns` | Date utilities |
| **clsx** | `npm install clsx` | Conditional class names |

---

**PICK at least 2 libraries per UI task** (1 animation + 1 icon minimum).
