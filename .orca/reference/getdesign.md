## 🎯 PURPOSE

Before coding ANY UI, consult real-world design systems.  
Pick 2-3 brands that match the project and study their patterns.

## TOP REFERENCE DESIGNS BY CATEGORY

- **Claude** — Warm, conversational, purple accents
- **Cursor** — Sleek dark, gradient accents, VS Code inspired
- **Mistral AI** — Clean blue, developer-focused
- **Ollama** — Terminal-inspired, minimal, green accent

### Developer Tools
- **Linear** — Ultra-minimal, purple, dark-first

1. Visit https://getdesign.md/ for full design system analysis

3. Extract: color palette, typography, spacing, component patterns, animation
4. Adapt (don't copy) — use their psychology, your execution
