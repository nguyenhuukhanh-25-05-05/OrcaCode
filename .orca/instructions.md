## ⚡ QUICK START

### For Simple Components (Button, Input):
```
1. Read: core/philosophy.md (2 min)
2. Check: design/colors.md + design/styles.md (1 min)
3. Code: Use components/examples.md as template (5 min)
4. Verify: qa/checklist.md (2 min)

Total: ~10 minutes
```

### For Complex Components (Form, Dashboard):
```
1. Read: core/philosophy.md + core/workflow.md (5 min)
2. Inspiration: reference/getdesign.md - pick 2-3 brands (3 min)
3. Design: Create tokens using design/* modules (5 min)
4. Code: Follow components/patterns.md (15 min)
5. Animate: Apply animation/rules.md (5 min)
6. Audit: qa/checklist.md + qa/performance.md (5 min)

Total: ~40 minutes
```

## 🚨 CRITICAL RULES (Never Skip)

### Rule 1: Read Philosophy First
**ALWAYS** read `core/philosophy.md` before any UI work.

### Rule 2: Follow Workflow
Use the 5-phase workflow from `core/workflow.md`:
1. Design Planning
2. Component Structure  
3. Animation Implementation
4. Performance Audit
5. Final Review

### Rule 3: Pass Quality Gate
Score **16/16** on `qa/checklist.md` before presenting work.

### Rule 4: Reference Real Designs
Check `reference/getdesign.md` for inspiration (2-3 brands minimum).
